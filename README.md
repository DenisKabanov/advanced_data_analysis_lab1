adwanced_data_analysis_lab1
==============================

В качестве основной метрики был выбран F_1 score по нескольким причинам:
1) так как в задаче предсказывания заболеваний нам важнее, сколько из реально правильных было предсказано правильно (то есть FN -> min) в метрике recall = TP/(TP+FN)
2) recall легко обманывается при помощи предсказания всех 1, поэтому правильнее будет использовать его сочентание с precision = TP/(TP+FP), то есть F_1 score

При воспроизведении, убедитесь, что корневая директория видна для python (путь есть в PATH или PYTHONPATH), чтобы он мог видеть модули.
